How This Model Works

The problem, in plain words

For a given country and crop in a given year, estimate that year's yield (hg/ha) from the current year's rainfall, pesticide use and temperature, plus the previous five years of history. Because it uses the current year's own conditions, this is now-casting.

The data & features

The FAO panel: 101 countries × 10 crops, 1990–2013. Each record is enriched with 5-year lag features for yield, rainfall, pesticides and temperature — so every prediction sees a short window of recent history alongside the current year.

Eight models, three strategies

  • Option A — classic tabular models on flattened features: Random Forest, XGBoost, MLP embeddings, and a stacking ensemble.
  • Option B / C — neural nets with an LSTM branch over the 5-year sequence + learned country/crop embeddings (B = bidirectional, C = unidirectional).
  • Hybrids — deep LSTM features fed into XGBoost regressors.

The neural architecture (B & C)

The LSTM nets pair a sequence branch with embeddings:

  • Sequence branch — LSTM (32→16) reads the 5×4 lag window (bi-directional in Option B, uni-directional in Option C).
  • Embeddings — country→16, crop→8 dimensions.
  • Current-year inputs (year, rain, pesticides, temp) concatenated, then Dense 128 → 64 → 1 with dropout, batch-norm & L2.

How it was trained & judged

  • Leakage-safe temporal split: train 1990–2007, val 2008–2010, test 2011–2013. Scalers fit on train only.
  • Safe label encoding handles unseen countries/crops via an <UNKNOWN> token.
  • All 8 models scored on the same held-out test set with RMSE, R², MAPE & tolerance accuracy.
